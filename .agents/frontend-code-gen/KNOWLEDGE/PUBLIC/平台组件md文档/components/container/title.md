## Title 标题栏
页面标题栏组件。

### 基础用法

基础的标题栏用法。

:::demo 使用 `title` 属性，来定义标题栏的标题文字。

```html
<tl-title qrcode="test" title="这是一个标题"></tl-title>
```
:::

### Attributes
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| title     | 标题文字   | string  |  -  |  -   |
| qrcode     | 二维码内容   | string |   - | - |
| is-fixed     | 是否置顶   | boolean    | — | false   |
