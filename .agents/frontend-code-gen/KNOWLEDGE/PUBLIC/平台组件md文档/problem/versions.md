# 我该依赖什么版本的 yjpl 前端包？

### DAP和非平台项目
`DAP` 的项目需要按照研发管理部发布的规范，和 `平台` 对应版本进行依赖。

`DAP` 对应 `平台` 版本(不带`^` 和 `~`)：



1. `DAP8.0` -> `7.2.0`/`7.2.0-SP`
2. `DAP9.1` -> `8.1.0`/`8.1.0-SP`
3. `DAP9.2` -> `8.1.2`/`8.1.2-SP`
4. `DAP9.3` -> `8.2.0`/`8.2.0-SP`
5. `DAP9.5` -> `8.5.0`/`8.5.0-SP`

对 `平台` 包需要使用以下 `npm仓库` 进行下载：

``` shell
npm config set registry=https://jtmaven.ygsoft.com/repository/ygsoft-npm/
```

规范可以查看 [公司组件发布库管理指南规范](https://pan.ygsoft.com/l/yFgQCt)

有疑问详询各项目组pmo同事。
